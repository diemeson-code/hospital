# 🏥 SGHSS — Sistema de Gestão Hospitalar e Serviços de Saúde

Sistema Front-end desenvolvido para gerenciamento básico de pacientes e consultas médicas.

---

# 📌 Sobre o Projeto

O SGHSS (Sistema de Gestão Hospitalar e Serviços de Saúde) é um sistema web desenvolvido com HTML, CSS e JavaScript, com foco em interfaces modernas, organização visual e experiência do usuário.

O projeto simula um ambiente hospitalar simples, permitindo o cadastro de pacientes e o agendamento de consultas médicas.

---

# 🎯 Objetivo

O objetivo do projeto é aplicar conceitos de desenvolvimento Front-end, lógica de programação e manipulação do DOM por meio da criação de um sistema hospitalar funcional.

---

# 🚀 Funcionalidades

## 🔐 Login

* Validação de campos obrigatórios
* Navegação para o dashboard
* Interface moderna e responsiva

---

## 📊 Dashboard

* Exibição da quantidade de pacientes
* Exibição da quantidade de consultas
* Cards informativos
* Navegação entre páginas

---

## 👨‍⚕️ Cadastro de Pacientes

* Cadastro de pacientes
* Validação de formulário
* Exibição da lista de pacientes
* Armazenamento no LocalStorage

---

## 📅 Agendamento de Consultas

* Cadastro de consultas médicas
* Seleção de médico
* Seleção de data e horário
* Exibição das consultas cadastradas
* Persistência de dados com LocalStorage
